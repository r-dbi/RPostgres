# Determine database type for R vector.

Determine database type for R vector.

## Usage

``` r
# S4 method for class 'PqConnection'
dbDataType(dbObj, obj, ...)

# S4 method for class 'PqDriver'
dbDataType(dbObj, obj, ...)
```

## Arguments

- dbObj:

  Postgres driver or connection.

- obj:
